# **Angry Birds Computer Edition**

## Описание
Порт классической игры **Angry Birds** на **Python** с использованием **pygame**.

## Установка дополнительных библиотек
Если вы хотите переработать игру, можете создать отдельный репозиторий на GitHub, взяв за основу эту игру. Но для начала установите зависимости, выполнив команду:

```bash
pip install -r requirements.txt
```
